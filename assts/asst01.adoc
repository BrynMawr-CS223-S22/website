= Add

Write a program, `add.c`, that takes two numbers (assume floats) as command line arguments 
and then prints their sum. A session trace for this program might be: 

[source]
----
$ **make add**
$ **./add 4 5**
4.0 + 5.0 = 9.0
$ **./add 40 -3.5** 
40.0 + -3.5 = 36.5      
----
